# Tasks App 📝

A simple Flutter app for managing tasks, built with:

- 🧠 BLoC for state management
- 💾 Sqflite for local database
- 🎯 Dart & Flutter

## Features

- Add, update, and delete tasks
- Store tasks locally
- Clean architecture with Cubit
